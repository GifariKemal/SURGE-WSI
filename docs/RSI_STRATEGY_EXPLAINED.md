### 2. RSI CALCULATION (Kode Python)

```python
def calculate_rsi(df, period=14):
    """
    Hitung RSI dari data OHLCV
    """
    # 1. Hitung perubahan harga
    delta = df['close'].diff()

    # 2. Pisahkan gain dan loss
    gain = delta.where(delta > 0, 0)    # Positif saja
    loss = -delta.where(delta < 0, 0)   # Negatif jadi positif

    # 3. Hitung rata-rata 14 periode
    avg_gain = gain.rolling(14).mean()
    avg_loss = loss.rolling(14).mean()

    # 4. Hitung RS dan RSI
    rs = avg_gain / (avg_loss + 0.0000001)  # Hindari bagi 0
    rsi = 100 - (100 / (1 + rs))

    return rsi
```

## Kenapa RSI Strategy Bekerja?

### 1. Mean Reversion adalah Real
Harga tidak bergerak satu arah selamanya. Setelah bergerak ekstrem, ada kecenderungan untuk kembali ke mean.

### 2. RSI Mengukur "Kelelahan" Pasar
- RSI < 30 = Seller sudah "habis", buyer mulai masuk
- RSI > 70 = Buyer sudah "habis", seller mulai masuk

### 3. London Session = Likuiditas Tinggi
- Volume trading tertinggi
- Spread lebih kecil
- Pergerakan lebih "clean"

### 4. Simple = Robust
- Sedikit parameter = sedikit overfitting
- Logika sederhana = bekerja di berbagai kondisi market
